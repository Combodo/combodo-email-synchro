The data model can be modified on the fly without reinstalling iTop or re-building the database. Just make sure that you adjust the database schema accordingly: for example if you change the possible values for an "enum", the MySQL DB schema must be updated. If you add a new field to an object, the DB schema must be updated. If you add new objects, the DB schema must be updated as well. Combodo has some tools to help you when doing such modifications, contact support@combodo.com if you want to benefit from them.
